# 申し送り（タブB→タブA）: 「業界紹介動画」専用タブ新設＋videotabset allowlist追加

## 依頼
動画は3種類になりました：会社の **企業紹介動画(L2)** / **決算書分析動画(L3)** に加え、**業界紹介動画（16業界×1本）** を生成中。
既存の `videotabset` は動画2タブ(企業紹介動画/決算書分析動画)のみ許可。**業界動画の視聴リンク＋FBを書く先が無い**ため、専用タブ新設をお願いします。

## お願いしたい対応
1. **新タブ「業界紹介動画」を作成**（列様式は動画2タブと同一：col1=業界名, col2=…, **col3=ステータス, col4=公開URL(視聴リンク), col6=FB1**, round毎に4列…col45=更新。会社タブの見本行3の書式を継承でOK）。
   - ★突合キーは**会社名ではなく「業界名」**（col1 or col2 に業界名）。16業界の行を事前作成いただけると確実（下記16業界名）。無ければ `append=1` で追加します。
2. **`videotabset` の allowlist に `業界紹介動画` を追加**（`sheet=業界紹介動画` を許可）。突合は業界名で（`_findRowByCompany` を業界名にも対応 or 業界名列で照合）。
   - タブB側は `videotabset` を `sheet=業界紹介動画 & company=<業界名> & url=<リンク> [&append=1&industry=<業界名>]` で叩きます（会社タブと同じI/F）。
3. deploy後、`sheethead&sheet=業界紹介動画` が通る状態（列様式返却）にしてください。→ タブB側watcherが自動でリンク記入を開始します。

### タブB側の呼び方（会社タブと同じ・`company`に業界名を渡す）
```bash
curl -sL "$WU?mode=videotabset&token=$ST" \
  --data-urlencode "sheet=業界紹介動画" \
  --data-urlencode "company=総合商社" \
  --data-urlencode "url=https://youtu.be/xxxxx" \
  --data-urlencode "fb=（任意）"
# → {"ok":true,"sheet":"業界紹介動画","company":"総合商社","row":4,"written":{"url_col":4,"fb_col":6}}
```
